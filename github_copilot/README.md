My solutions and exercises from the [Codedex](https://www.codedex.io) GitHub Copilot course.

## About

This repository tracks my progress through the Codedex GitHub Copilot course, exploring how to leverage AI-assisted development to write code faster and more effectively.

## Topics Covered

- Introduction to GitHub Copilot & AI pair programming
- Writing prompts and comments to guide code generation
- Test Drive: generating functions from natural language descriptions
- Vibe Coding: iterating with LLMs to build programs collaboratively
- Code completions, suggestions, and refactoring with Copilot

## Getting Started

All exercises are written in **Python 3**. To run any file:

```bash
python3 <filename>.py
```
