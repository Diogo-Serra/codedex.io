My solutions and exercises from the [Codedex](https://www.codedex.io) Command Line course.

## About

This folder tracks my progress through the Codedex Command Line course, learning how to navigate and control a computer entirely through the terminal.

## Topics Covered

- Introduction to the Terminal & Shell
- Navigating the file system (`cd`, `ls`, `pwd`)
- Managing files & directories (`touch`, `mkdir`, `cp`, `mv`, `rm`)
- Viewing & editing file contents (`cat`, `less`, `nano`, `echo`)
- Permissions & ownership (`chmod`, `chown`)
- Piping & redirection (`|`, `>`, `>>`, `<`)
- Environment variables & the `$PATH`
- Shell scripts and automation (`bash`, `#!/bin/bash`)

## Getting Started

No installation needed — just open a terminal. To check your shell:

```bash
echo $SHELL
```
